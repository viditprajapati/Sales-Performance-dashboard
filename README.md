This project presents a Sales Analysis Dashboard built using Power BI to analyze business performance across multiple dimensions such as time, products, cities, and promotions.

The dashboard provides actionable insights into sales trends, profitability, product performance, and promotional effectiveness, helping stakeholders make data-driven decisions.

#Objectives
Analyze sales trends from 2020 to 2024
Compare profit vs net sales
Identify top and bottom performing products
Evaluate promotion effectiveness
Understand city-wise sales distribution
🛠️ Tech Stack
Power BI Desktop – Data visualization & dashboard creation
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – KPI calculations and measures
Data Modeling – Relationships between tables (Sales, Products, Customers, Promotions)

#Data Description

The dataset consists of transactional sales data including:

Order ID, Customer ID, Date
Product ID, Product Name
Sales, Net Sales, Profit
Discount & Discount Percentage
Promotion ID & Promotion Name
Units Sold & Price per Unit

The data is structured to support filtering and aggregation across multiple dimensions.
